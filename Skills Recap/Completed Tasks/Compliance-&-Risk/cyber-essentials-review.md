# Cyber Essentials Controls Review Guide

## 1. Identify the Scope

-   Define which systems, devices, and networks are in scope.
-   Include remote workers and cloud services.

## 2. Gather Evidence

-   Network diagrams
-   Firewall screenshots
-   Patch reports
-   Anti-malware dashboards
-   Access control lists

## 3. Review Each Control Area

### 3.1 Firewalls & Internet Gateways

-   Only required ports open.
-   Deny-by-default rules.
-   No default passwords.
-   Firmware up to date.

### How to Review

-   Use external scans.
-   Inspect firewall configuration.
-   Validate admin interface restrictions.

------------------------------------------------------------------------

### 3.2 Secure Configuration

-   Remove unnecessary software.
-   Disable auto-login.
-   Configure baseline builds.

### How to Review

-   Check AD/GPO baselines.
-   Review build templates.
-   Validate removal of local admin rights.

------------------------------------------------------------------------

### 3.3 User Access Control

-   MFA enabled.
-   Least privilege.
-   Leaver accounts removed promptly.

### How to Review

-   Audit AD groups.
-   Check admin account usage.
-   Review MFA status.

------------------------------------------------------------------------

### 3.5 Patch Management

-   Patches applied within 14 days.
-   Unsupported software removed.
-   Automatic updates enabled.
